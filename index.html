// PROJECT 01

ACB Installation & Power Distribution Optimization

Redesigning power flow architecture to eliminate cascading trip faults and improve load resilience

Power Systems
// The Problem

The facility was experiencing recurring cascade trips across distribution panels, causing unplanned full-plant shutdowns averaging 4 hours per event. The existing Air Circuit Breakers were undersized for actual load demand, protection relay coordination was absent, and there was no discriminative selectivity between upstream and downstream breakers. Each fault was propagating upstream instead of isolating at the nearest protection point.

// Approach & Execution
  • Conducted full load audit and fault level calculation using ETAP
  • Designed ACB rating and protection relay coordination scheme
  • Specified and installed correctly rated ACBs with LSIG protection
  • Programmed zone-selective interlocking (ZSI) between breakers
  • Implemented bus-tie arrangement for redundancy during maintenance
  • Commissioned with phased energisation and protection testing
// Quantified Results
100% Elimination of cascade trips post-commissioning
4hrs Average outage duration eliminated per event
↑37% Improvement in overall production uptime
0 Arc flash incidents since commissioning
ETAP ACB / LSIG Protection Zone Selective Interlocking Protection Relay Coordination AutoCAD Electrical Load Flow Analysis
// PROJECT 02

Automatic Voltage Regulator (AVR) System Implementation

Deploying closed-loop voltage stabilisation to protect sensitive production equipment from supply fluctuations

Power Quality
// The Problem

Voltage fluctuations from the utility supply — regularly swinging ±15% of nominal — were causing erratic behavior in variable speed drives, PLC processor resets, and premature motor insulation failure. The maintenance team was replacing motors at twice the expected lifecycle, and production stoppages from equipment trips were averaging 6 events per month. The root cause was unregulated incoming voltage feeding directly into sensitive automation panels.

// Approach & Execution
  • Installed power quality analyser to characterise voltage profile over 30 days
  • Specified AVR units rated to handle recorded fluctuation envelope
  • Designed AVR placement strategy: upstream of critical automation loads
  • Integrated AVR status signals into existing SCADA for real-time alarm
  • Established voltage monitoring KPIs in CMMS for trending
  • Trained maintenance team on AVR operation and bypass procedures
// Quantified Results
−85% Reduction in voltage-related equipment trips
2× Extension of average motor service life
30% Drop in unplanned downtime events (monthly)
↓25% Reduction in electrical maintenance spend
Power Quality Analysis AVR Specification & Installation SCADA Integration CMMS — IFS Motor Protection VSD Systems
// PROJECT 03

PLC/SCADA Automation & Process Optimisation

Replacing manual process control with intelligent automation to achieve consistent throughput and 25% energy reduction

Industrial Automation
// The Problem

A production line operating on legacy relay-based control logic was bottlenecked by manual intervention at every stage. Operators were making real-time decisions based on experience rather than data, causing inconsistent cycle times and energy consumption peaks during changeovers. There was no centralised visibility into machine states, no alarm logging, and fault diagnosis required a specialist every time — adding 2–3 hours per fault event in lost production time.

// Approach & Execution
  • Mapped existing relay logic and documented full process sequence
  • Migrated logic to Siemens S7-300 PLC with structured function blocks
  • Designed HMI screens for operator visibility and manual override
  • Built SCADA dashboards: real-time OEE, energy consumption, fault log
  • Programmed automatic sequencing to eliminate manual intervention points
  • Implemented alarm management system with prioritisation matrix
  • Ran 30-day parallel operation before full cutover
// Quantified Results
25% Energy consumption reduction through optimised sequencing
−70% Fault diagnosis time (2–3hrs → under 30 mins)
↑18% Throughput increase from consistent cycle control
100% SCADA visibility across all critical process nodes
Siemens S7-300 PLC TIA Portal SCADA / WinCC HMI Design OEE Monitoring Alarm Management Energy Monitoring
